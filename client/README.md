## 👨‍💻 Como executar ?

Executar os seguintes códigos na pasta desejada:

- git clone https://github.com/gdpnei2002/Solar-Amazon.git
- cd Solar-Amazon
- npm install
- yarn start

Ou você pode conferi-lo online:

https://solar-amazon.vercel.app
